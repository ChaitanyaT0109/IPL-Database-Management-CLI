# IPL - Database Management System

## Viewer Mode
### 1. League
#### League Options
- *Matches*: Displays a list of matches in the selected league, including match ID, date, and format.
- *League Information*: Retrieves and displays detailed information about the selected league.
- *Teams Information*: Shows information about all teams participating in the league.
- *Achievements*:
  - *Players with Most Runs*: Lists the top five players with the highest runs scored in the league.
  - *Players with Most Wickets*: Lists the top five players with the most wickets taken in the league.
- *Team Standings*: Displays the current standings of teams in the league, including matches played, won, lost, and points.
- *Back to Main Menu*: Returns to the main viewer menu.

---

### 2. Match
#### Match Options
- *Match Information*: Provides details of all matches, including match ID, date, time, and format.
- *Scorecard*: Allows users to view team statistics for a specific match based on match ID, including runs, wickets, and best player performances.
- *Highlights*: Fetches and displays highlights for all matches.
- *Back to Main Menu*: Returns to the main viewer menu.

---

### 3. Player
#### Player Options
- *Player Statistics*: Displays comprehensive statistics for all players, including runs scored and wickets taken.
- *Search Player*: Enables users to search for players by entering a substring of their names and retrieves matching records.
- *Back to Main Menu*: Returns to the main viewer menu.

---

### 4. Venue
#### Venue Options
- *Matches Played*: Lists all matches held at various venues along with stadium names and locations.
- *Venue Popularity*: Shows popularity rankings of stadiums based on the number of matches played at each venue.
- *Back to Main Menu*: Returns to the main viewer menu.

---

### 5. Exit
- Exits the viewer mode.

---

## Administrator Mode
### 1. Insert
#### Insert Menu
- *Insert into Player*: Allows administrators to add new player details into the database.
- *Insert into Stadium*: Enables administrators to add new stadium details into the database.
- *Back to Administrator Menu*: Returns to the administrator menu.

---

### 2. Update
#### Update Menu
- *Update Stadium*: Facilitates updating existing stadium details in the database.
- *Back to Administrator Menu*: Returns to the administrator menu.

---

### 3. Exit
- Exits the administrater mode.

---

## Exit mode
- Exits the Application
